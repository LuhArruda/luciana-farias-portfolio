# 🚀 Portfólio Profissional | Luciana Farias

Bem-vindo ao meu repositório! Este projeto é um portfólio moderno desenvolvido para destacar minhas competências em **Desenvolvimento Low-Code (OutSystems)** e **Front-end (React.js)**.

## 🛠️ Tecnologias
- **React.js + Vite** (Performance e agilidade)
- **Tailwind CSS** (Estilização Utility-First)
- **Wouter** (Roteamento leve e eficiente)
- **Lucide React** (Ícones minimalistas)

## 🌟 Destaques Técnica
- **Navegação Dinâmica:** Sistema de rotas otimizado para detalhes de serviços.
- **UI/UX:** Interface focada em conversão e experiência do usuário.
- **Clean Code:** Organização de componentes seguindo as melhores práticas.

*Desenvolvido com foco em excelência técnica e inovação.*
